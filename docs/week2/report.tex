\section{Implementation}

\subsection{Regard Classifier}
The \texttt{NLGBiasClassifier} in \texttt{src/app/scoring.py} loads the
\texttt{sasha/regardv3} model, handles device placement, and provides batch
prediction with four output labels (negative, neutral, positive, other).

\subsection{Demographic Masking}
The \texttt{mask\_text()} function replaces all case variants of demographic
terms with ``XYZ'' before scoring, isolating content-based bias from explicit
demographic mentions.

\subsection{Baseline Metrics}
\begin{itemize}
\item \texttt{compute\_regard\_distribution()}: Regard proportions per demographic
\item \texttt{compute\_negative\_regard\_gap()}: $\Delta_{\text{neg}} = P(\text{neg}|A) - P(\text{neg}|B)$ per prompt type
\item \texttt{compute\_bootstrap\_ci\_for\_gap()}: 95\% bootstrap CIs for gaps
\end{itemize}

\subsection{Sanity Checks}
\begin{itemize}
\item Label distribution verification (all labels present, sufficient samples)
\item Empty and short completion checks
\item Spot-check samples (20 outputs balanced across labels)
\end{itemize}
